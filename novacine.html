Mapeo manual de URLs

Para títulos que los servidores automáticos no encuentran (ej. Doraemon en español). Si una película/serie tiene una URL aquí, se usará en lugar de los servidores estándar.

Tip: Si la URL contiene {s} y {e}, se reemplazan por temporada y episodio.
Ej. https://miembed.com/tv/50/{s}/{e}

Bloquear anuncios con DNS

Los servidores embed usan dominios de anuncios. Cambiar tu DNS bloquea esos anuncios en TODO el dispositivo (todas las apps y navegadores), incluyendo dentro de los iframes. Es la solución más efectiva.

NextDNS (configurable)
Crea cuenta gratis en nextdns.io → activa "Ads & Trackers" + "Adult content optional".

Cómo aplicarlo:
• Android: Ajustes → Conexiones → DNS privado → Hostname → pega DoH.
• iPhone: Instala perfil de AdGuard desde adguard-dns.io.
• Windows 10/11: Settings → Network → Ethernet/Wi-Fi → Edit DNS → pega IPv4.
• Router (recomendado): cambia DNS en el router para que afecte a toda la red.
